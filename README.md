# CUISLE
### Cardiovascular Universal Instrument for Shared-source Low-cost Experimentation

*Cuisle (kwee-shla) — Irish for pulse/vein/artery*

---

A low-cost, open-source instrument for versatile vascular testing, developed at Trinity College Dublin. CUISLE supports four testing configurations — ring, wire myography, pressure myography and uniaxial — from a single multifunctional attachment, at a unit cost of €269.38 (inc. VAT, Ireland, March 2026).

---

| Type | Description |
